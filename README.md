# 3-Tier Web Application

## Overview
This 3-Tier Web Application is designed to demonstrate the architecture and functionalities of a web app that separates presentation, application processing, and data management.

## Architecture
- **Presentation Tier:** This layer includes the user interface and facilitates communication between the user and the application. Technologies used: HTML, CSS, JavaScript.
- **Application Tier:** This layer contains the business logic and processing, handling requests from the presentation tier. Technologies used: Node.js, Express.
- **Data Tier:** This layer consists of the database where data is stored and managed. Technologies used: MongoDB.

## Features
- User Authentication
- RESTful API
- Responsive Design
- Data Visualization

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yesk993-ops/3-tier-web-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd 3-tier-web-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage
- Open a web browser and go to `http://localhost:3000` to view the application.
